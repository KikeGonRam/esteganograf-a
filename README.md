# Proyecto Avanzado de Esteganografía

Este proyecto implementa técnicas avanzadas de esteganografía en imágenes, audio y video, incluyendo utilidades de cifrado y una interfaz de línea de comandos (CLI).

## Estructura

- `src/`: Código fuente principal
- `tests/`: Pruebas unitarias y de integración
- `data/`: Archivos de ejemplo para pruebas
- `docs/`: Documentación técnica y de usuario
- `scripts/`: Utilidades y scripts auxiliares

## Requisitos

- Python 3.12 (entorno conda)
- Dependencias en `requirements.txt`
